# ML Engineer Portfolio & Sentiment Analyzer ⚡

A high-performance portfolio app featuring a live **RNN vs LSTM** sentiment analysis comparison and details on my **RAG Pipeline** and **CNN** projects.

## 🔗 [Live Demo Link](https://rafi-veyis-portfolio-ui.streamlit.app/)

### 🚀 Key Features
* **Live Inference**: Compare RNN and LSTM models on custom text.
* **Architecture Visuals**: Deep dives into my 95% accuracy CNN model.
* **Interactive UI**: Custom-styled with Syne & Space Mono for a "Cyberpunk" aesthetic.

### 🛠️ Tech Stack
* **Models**: PyTorch (RNN, LSTM, CNN)
* **LLM**: LangChain (RAG Pipeline)
* **Frontend**: Streamlit + Custom CSS/HTML
